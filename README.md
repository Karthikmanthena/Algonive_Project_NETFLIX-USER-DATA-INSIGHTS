Project Overview

This project is an interactive Streamlit dashboard built to analyze Netflix content data and extract insights related to content trends, user preferences, and recommendation strategies.
It combines data cleaning, exploratory data analysis (EDA), clustering, and content-based recommendations in a single web application.

The dashboard helps understand:

Growth of Netflix content over time

Popular genres and regions

Differences between Movies and TV Shows

Similar content recommendations using data-driven logic
